مرجع سریع برنامه‌نویسی اندروید زیر چاپ

نام کتاب مرجع سریع برنامه‌نویسی اندروید
سال انتشار 2015
نویسنده اونور چنار
مترجم دکتر قاسم کیانی مقدم
شابک 978-964-377-844-6

خرید از وب‌سایت انتشارات ناقوس

در باره‌ی این کتاب

امروزه دیگر اندروید صرفاً سیستم عاملی برای دستگاه‌های موبایل نیست، بلکه دستگاه‌های ارتباطی مختلفی، از قبیل تلویزیون، و دستگاه‌های پوشیدنی نیز از آن استفاده می‌کنند. پلتفرم اندروید با رابط برنامه‌نویسی گسترده‌ی خود فرصت‌های بی‌پایانی را برای توسعه دهندگان پدید می‌آورد.

مرجع سریع برنامه‌نویسی اندروید مرجع فشرده‌ای برای برنامه‌نویسی و API اندروید است، از جمله APIهای جدید که در اندروید آب‌نبات چوبی 5.0 معرفی شده‌اند. این کتاب رابط‌های برنامه‌نویسی اساسی اندروید را به طور سازمان یافته‌ای ارائه می‌کند که می‌توان راحت و سریع از آن استفاده کرد. این کتاب مملو از اطلاعات مفید است و مرجعی ضروری برای تمام توسعه دهندگان و برنامه‌نویسان اپلیکیشن‌های اندروید است.

چیزهایی که در این کتاب خواهید آموخت:

  • مقدمه‌ای کوتاه بر پلتفرم اندروید و محیط توسعه‌ی آن.
  • بخش‌های اساسی اپلیکیشن‌های اندروید، از قبیل کامپوننت‌های رابط کاربری، خبررسانی‌ها، و منابع.
  • ذخیره‌سازی و دسترسی داده‌ها با استفاده از API اندروید.
  • دسترسی به مکان، و استفاده از حسگرهای دستگاه.
  • ضبط و پخش محتوای صوتی و ویدئویی، و دسترسی به دوربین.
